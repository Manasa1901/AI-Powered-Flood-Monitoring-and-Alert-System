# 🌊 Smart City Flood Monitoring System

## 📌 Project Overview

The Smart City Flood Monitoring System is a web-based application designed to monitor, predict, and manage flood situations in urban areas. It integrates weather data, flood reports, and machine learning techniques to provide real-time flood risk assessment and emergency alerts. The system helps citizens stay informed and enables authorities to take preventive actions during flood emergencies.

---

## 🚀 Features

- Real-time flood monitoring dashboard
- Flood incident reporting by citizens
- Interactive map visualization
- Machine Learning-based flood prediction
- Emergency alerts and notifications
- Historical flood data tracking
- Weather data integration
- Emergency contact information

## 📊 Project Workflow

1. Users report flood incidents through the application.
2. Weather and environmental data are collected.
3. The backend processes incoming data.
4. Data is stored in the MySQL database.
5. The machine learning model analyzes flood risk.
6. Alerts are generated for high-risk areas.
7. Notifications are sent to users.
8. Dashboard updates in real time.
